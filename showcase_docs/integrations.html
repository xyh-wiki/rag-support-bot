Integrations

Slack

The Slack integration posts indexing failures, weekly usage summaries, and billing alerts to a selected channel. It requires the chat:write and channels:read scopes.

GitHub

The GitHub integration can import Markdown files from a repository. It supports branch selection, path filters, and scheduled sync every 6 hours.

Notion

The Notion integration imports selected pages and databases. Synced pages keep their original Notion URL as source metadata.

Zapier

Zapier actions can create documents, invite members, and trigger workspace exports. Zapier polling runs every 5 minutes on paid Zapier plans.
