# Getting Started

## What is Northwind Docs Hub

Northwind Docs Hub is a fictional SaaS platform used as a sample knowledge base. It helps teams publish internal documentation, connect third-party systems, and search support articles from a browser widget.

The sample workspace includes three plans: Starter, Pro, and Business. Starter is for small teams, Pro adds automation and higher API limits, and Business adds SSO, audit logs, and contractual support.

## Create a workspace

To create a workspace, sign in with an email address, choose a workspace name, and select a data region. The available regions are United States, European Union, and Singapore.

Workspace names can contain letters, numbers, hyphens, and spaces. The public workspace slug is generated from the name and can be changed once every 30 days.

## Invite teammates

Workspace owners can invite teammates from Settings -> Members. Invitations expire after 7 days. If an invitation expires, resend it from the pending invitations table.

Roles are Owner, Admin, Editor, Viewer, and Billing Manager. Only Owners can delete a workspace.

## Import documents

The import page accepts Markdown, PDF, Word, Excel, HTML, and plain-text files. A single import job can upload up to 200 files or 500 MB, whichever comes first.

Large PDF files are processed page by page. Excel files are indexed by worksheet and row. If a file contains no readable text, upload an OCR version.
